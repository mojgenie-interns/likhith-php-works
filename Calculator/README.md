# PHP CLI Calculator

A command-line calculator built using PHP that performs multiple arithmetic operations and runs continuously until the user exits.

---

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Power (^)
- Input validation
- Continuous execution loop

---

## Technologies Used

- PHP (CLI)
- VS Code

---

## How to Run

1. Open terminal  
2. Navigate to the project folder  

```bash
cd Calculator
php calculator.php
